Configuration des vlans

1ère étape : Création des vlans

On tape les commandes suivants pour créer et donner un nom aux différents vlans du réseaux :

enable => conf t => vlan 10 => name Finance

enable => conf t => vlan 20 => name RH

enable => conf t => vlan 30 => name Public

enable => conf t => vlan 40 => name Admin


2ème étape : Configuration des ports

Pour assigner les ports à chaque vlans du réseau on tape les commandes suivants :
