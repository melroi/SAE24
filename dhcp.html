Mise en place du service DHCP


Configuration IP du serveur linux

On fait la configuration ip du serveur linux dans le fichier /etc/network/interfaces, on reproduit la capture suivante :

Sur cette capture, on attribue sur la carte réseau "enp2s0", une adresse IP, un masque de sous-réseau, une passerelle et le DNS du réseau.

Ensuite on redémarre le serveur pour appliquer la configuration à l'aide de la commande "reboot".

Ensuite on installe le service dhcp à l'aide la commande "apt install dhcpd"

Puis on modifie le fichier /etc/default/isc-dhcp-server pour indiquer sur qu’elle carte on applique le DHCP

Pour continuer on modifie le fichier /etc/dhcp/dhcpd.conf pour donner les plages d’adresse qu’y serons donnée par le DHCP (penser à lui donner une plage d’adresse de son réseau sinon le DHCP ne marche pas)
